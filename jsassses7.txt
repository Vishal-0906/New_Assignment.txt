Q3. Create an HTML page with a button labeled 'Add Song' — when the button is clicked, use onclick to instantly add a new card below showing a random song title from your favorite playlist.

`html
<!DOCTYPE html>
<html>
<head><title>Add Song</title></head>
<body>
  <button onclick="addSong()">Add Song</button>
  <div id="song-list"></div>

  <script>
    const songs = ["Song A", "Song B", "Song C", "Song D"];
    function addSong() {
      const randomSong = songs[Math.floor(Math.random() * songs.length)];
      const card = document.createElement("div");
      card.textContent = randomSong;
      card.style.border = "1px solid #ccc";
      card.style.padding = "10px";
      card.style.margin = "5px";
      document.getElementById("song-list").appendChild(card);
    }
  </script>
</body>
</html>
`

Q10. Display a list of 3 movies (hardcoded) and add a 'Remove' button next to each. Use addEventListener to remove the movie card from the DOM when its button is clicked.

`html
<!DOCTYPE html>
<html>
<head><title>Movie List</title></head>
<body>
  <div id="movies"></div>

  <script>
    const movies = ["Inception", "Interstellar", "The Dark Knight"];
    const movieContainer = document.getElementById("movies");

    movies.forEach(movie => {
      const card = document.createElement("div");
      card.textContent = movie + " ";
      const removeBtn = document.createElement("button");
      removeBtn.textContent = "Remove";
      removeBtn.addEventListener("click", () => card.remove());
      card.appendChild(removeBtn);
      movieContainer.appendChild(card);
    });
  </script>
</body>
</html>
`

Q15. Briefly explain (in 2–3 sentences) what event flow is in the DOM, and give one example of how event bubbling could affect a shopping cart app like Flipkart.

Answer:

Event flow in the DOM describes how events propagate through elements in a hierarchy. It has two main phases: capturing (from the root down to the target) and bubbling (from the target back up to the root).  

For example, in a Flipkart shopping cart app, if a "Remove Item" button inside a product card is clicked, the event might bubble up to the parent container. If the parent also has a click handler (like removing the entire cart), both could trigger unintentionally unless event bubbling is stopped with event.stopPropagation().
