Accessing List
Theory question
Q1. Understanding how to create and access elements in a list?
A. A list is created using square brackets [].
 Elements are accessed using index positions starting from 0.

Q2. Indexing in lists (positive and negative indexing)?
A. Positive indexing starts from 0 from the beginning.
 Negative indexing starts from -1 from the end of the list.

Q3. Slicing a list: accessing a range of elements?
A. Slicing is used to access multiple elements using list[start:end].
 It returns a new list containing the selected range.

2. List Operations
Theory question

Q4. Common list operations: concatenation, repetition, membership?
A. Concatenation joins lists using +.
Repetition repeats a list using *.
Membership checks elements using in or not in.

Q5. List methods: append(), insert(), remove(), pop()?
A. append() adds an element at the end.
insert() adds at a specific index.
remove() deletes a value.
pop() removes an element by index.

4. Tuple
Theory question

Q9. Introduction to tuples, immutability?
A. A tuple is an ordered collection of elements.
Tuples are immutable, so values cannot be changed.

Q10. Creating and accessing elements in a tuple
A. Tuples are created using parentheses ().
Elements are accessed using index values.

Q11. Basic operations with tuples
A. Tuple operations include concatenation, repetition, and membership.
These operations do not change the tuple.

6. Dictionaries
Theory question

Q14. Introduction to dictionaries: key-value pairs
A. A dictionary stores data as key-value pairs.
Keys are unique and values can be any type.

Q15. Accessing, adding, updating, and deleting dictionary elements
A. Elements are accessed using keys.
Values can be added, updated, or deleted using keys.

Q16. Dictionary methods: keys(), values(), items()
A. keys() returns all keys.
values() returns all values.
items() returns key-value pairs.

Theory question

Q17. Iterating over a dictionary using loops
A. Dictionaries are iterated using for loops.
Keys, values, or both can be accessed.

Q18. Merging two lists into a dictionary
A. Two lists can be merged using loops or zip().
One list becomes keys and the other values.

Q19. Counting occurrences of characters in a string
A. Each character is stored as a key.
Its count is stored as the value in the dictionary.

A. Functions are defined using the def keyword.

Q21. Types of functions
A. Functions can be with or without parameters.
Functions can be with or without return values.

Q22. Anonymous functions (lambda functions)
A. Lambda functions are single-line functions.
They are defined using the lambda keyword.

9. Modules
Theory question

Q23. Introduction to Python modules and importing modules
A. Modules are files containing Python code.
They are imported using the import statement.

Q24. Standard library modules: math, random
A. math provides mathematical functions.
random is used to generate random values.

Q25. Creating custom modules
A custom module is a user-defined Python file.
It can be imported like a built-in module.
