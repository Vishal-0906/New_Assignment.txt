Q1
Question: Given an array of Flipkart product objects with properties name, price, and isInStock, use the filter() method to create a new array containing only the products that are in stock.  

Answer:
`javascript
const products = [
  { name: "Shoes", price: 2000, isInStock: true },
  { name: "Watch", price: 5000, isInStock: false },
  { name: "Bag", price: 1500, isInStock: true }
];

const inStockProducts = products.filter(product => product.isInStock);
console.log(inStockProducts);
`

Q7
Question: Given an array of Zomato order objects with a clicks property, use reduce() to calculate the total number of clicks across all orders.  

Answer:
`javascript
const orders = [
  { id: 1, clicks: 10 },
  { id: 2, clicks: 25 },
  { id: 3, clicks: 15 }
];

const totalClicks = orders.reduce((sum, order) => sum + order.clicks, 0);
console.log("Total Clicks:", totalClicks);
`

Q13
Question: Ask ChatGPT to suggest a one-liner using map() and reduce() that calculates the sum of all prices in an array of Myntra cart items, and then implement the suggested code in your editor.  

Answer:
`javascript
const cartItems = [
  { name: "T-shirt", price: 500 },
  { name: "Jeans", price: 1200 },
  { name: "Shoes", price: 2000 }
];

// One-liner using map() + reduce()
const totalPrice = cartItems.map(item => item.price).reduce((sum, price) => sum + price, 0);
console.log("Total Price:", totalPrice);
`
