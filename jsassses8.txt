Q12. Given an array of objects called movies, where each object has properties: title, year, and a nested object rating with keys imdb and rottenTomatoes, write code to print each movie's title and its IMDb rating.

`javascript
let movies = [
  { title: "Inception", year: 2010, rating: { imdb: 8.8, rottenTomatoes: 87 } },
  { title: "Interstellar", year: 2014, rating: { imdb: 8.6, rottenTomatoes: 72 } },
  { title: "The Dark Knight", year: 2008, rating: { imdb: 9.0, rottenTomatoes: 94 } }
];

movies.forEach(movie => {
  console.log(${movie.title} - IMDb Rating: ${movie.rating.imdb});
});
`

Q13. Create an array called orders, where each order is an object with properties: orderId, items (an array of objects with itemName and price), and delivery (an object with address and status). Write a loop to display each order's orderId and total price of all items.

`javascript
let orders = [
  {
    orderId: 101,
    items: [
      { itemName: "Shoes", price: 2000 },
      { itemName: "Socks", price: 200 }
    ],
    delivery: { address: "Vadodara", status: "Delivered" }
  },
  {
    orderId: 102,
    items: [
      { itemName: "Watch", price: 1500 },
      { itemName: "Chain", price: 800 }
    ],
    delivery: { address: "Ahmedabad", status: "Pending" }
  }
];

orders.forEach(order => {
  let total = order.items.reduce((sum, item) => sum + item.price, 0);
  console.log(Order ID: ${order.orderId}, Total Price: ₹${total});
});
