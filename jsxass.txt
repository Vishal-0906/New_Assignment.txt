Task 9
窶飢uestion:
Modify your insta-bio.html file to add a <button> labeled 'Show Greeting'. When the button is clicked, use JavaScript to show an alert that says 'Hello from JS!'. Hint: Use document.querySelector and addEventListener to connect the button with your JS code.
窶帰nswer:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Insta Bio</title>
</head>
<body>
    <h1>My Bio Page</h1>
    <button id="greetBtn">Show Greeting</button>

    <script>
        // Select the button element by its ID
        const targetBtn = document.querySelector('#greetBtn');
        
        // Listen for a click event to fire the alert
        targetBtn.addEventListener('click', function() {
            alert('Hello from JS!');
        });
    </script>
</body>
</html>

Task 10
窶飢uestion:
Edit your HTML file to include an <h2> tag with the text 'Top 3 Food Apps'. Use JavaScript in a <script> tag to change the color of this heading to green when the page loads. Hint: Use document.querySelector to select the heading and style.color to change its color.

Answer:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Food App Heading Styling</title>
</head>
<body>
    <h2 id="food-heading-title">Top 3 Food Apps</h2>

    <script>
        // Select the heading using querySelector
        const appHeading = document.querySelector('#food-heading-title');
        
        // Apply inline styles via JS to alter color properties
        appHeading.style.color = 'green';
    </script>
</body>
</html>
