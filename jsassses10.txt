Q13. Refactor this function to use an arrow function and rest parameters: function addToCart() { let total = 0; for(let i=0; i<arguments.length; i++) { total += arguments[i]; } return total; } The new function should be called addToCartArrow and accept any number of prices as arguments, returning their sum.

`javascript
const addToCartArrow = (...prices) => {
  return prices.reduce((total, price) => total + price, 0);
};

console.log(addToCartArrow(100, 200, 300));
`

Q15. Write an arrow function called getFollowersSummary that takes any number of follower counts (using the rest operator), sums them, and returns a string like 'Total followers: 1200'. Call it with 3 sample numbers to test.

`javascript
// Arrow function using rest operator
const getFollowersSummary = (...counts) => {
  const total = counts.reduce((sum, count) => sum + count, 0);
  return Total followers: ${total};
};

// Calling with 3 sample numbers
console.log(getFollowersSummary(300, 400, 500));
